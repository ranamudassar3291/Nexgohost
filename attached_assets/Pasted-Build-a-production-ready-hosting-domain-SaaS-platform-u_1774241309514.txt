Build a production-ready hosting & domain SaaS platform using Node.js (Express) + MongoDB.

========================
1. DOMAIN TRANSFER (REAL REGISTRAR API)
========================

Integrate real registrar API (Namecheap or ResellerClub).

Flow:

1. User enters domain
2. System calls API:
   - Check domain exists
   - Check transfer eligibility
   - Check transfer lock status
3. Show:
   - Domain status
   - Lock status (Locked / Unlocked)

4. Connect TLD pricing page:
   - Fetch transfer price dynamically from TLD pricing table
   - Example:
     .com → $8.99
     .net → $10.99

5. Validate EPP code:
   - Send to registrar API

6. On success:
   - Create order
   - Generate invoice
   - Status = Pending Transfer

7. Save domain in database:
   - domain
   - status
   - lockStatus
   - transferStatus
