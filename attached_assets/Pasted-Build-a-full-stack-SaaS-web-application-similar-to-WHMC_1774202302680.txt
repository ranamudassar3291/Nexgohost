1. DOMAIN TRANSFER SYSTEM (REAL-LIKE FLOW)
========================

Create a domain transfer feature with the following logic:

- User enters domain name
- System performs:
  1. Check if domain is valid format
  2. Simulate LIVE API check:
     - Check if domain exists
     - Check transfer lock status (locked/unlocked)
     - Check EPP code validity (fake vs real simulation logic)
